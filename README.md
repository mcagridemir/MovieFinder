# MovieFinder

you want to find a movie you'd like, MovieFinder is all yours :)

A sample app to experiment with:
- MVVM
- Firebase(Analytics, Push Notification)
- Moya&Alamofire
- SDWebImage
- SwiftGen
- SwiftLint
- SkeletonView
